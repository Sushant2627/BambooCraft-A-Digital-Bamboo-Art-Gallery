# 🎨 BambooCraft – A Digital Bamboo Art Gallery

The **BambooCraft – A Digital Bamboo Art Gallery** is a web-based application developed using **PHP and MySQL** to manage art products, users, enquiries, and purchases efficiently.  
It allows art galleries to showcase artworks online while enabling users to browse, search, and interact with available art pieces.


---

## 🚀 Features

### 👤 User Module
- User registration and login
- Browse artworks
- Search artworks
- View detailed artwork information
- Add artworks to cart
- Submit art enquiries
- Secure logout

### 🛒 Cart & Purchase
- Add items to cart
- View cart details
- Simple purchase workflow

### 🛠 Admin Module
- Admin authentication
- Manage art products
- View and manage user enquiries
- Maintain gallery data

---

## 🧰 Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Bootstrap  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP (Apache & MySQL)  
- **Version Control:** Git & GitHub

## 🗄 Database Setup

1. Open **phpMyAdmin**
2. Create a database named:

agmsdb
3. Import the file:

agmsdb.sql
4. Update database credentials in the `includes` folder if required.

---

## ▶ How to Run the Project

1. Install **XAMPP**
2. Start **Apache** and **MySQL**
3. Copy the project folder to:
C:\xampp\htdocs\agms

4. Open your browser and visit:
http://localhost/agms
